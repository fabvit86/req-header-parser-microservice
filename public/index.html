Request Header Parser Microservice

USAGE: Visiting https://fabvit86-reqheadparser-ms.herokuapp.com/parseheader will return the IP address, language and operating system of the client in JSON format.

For example, a typical respose could be: {"language":"it-IT","ip":"87.21.112.169","client_OS":"Apple Mac; macOS Sierra"}.
